# TreasureGame
Find the treasure the PC has hidden!



Note your choice to the 'Where do you think the treasure is?'
must be a 2 digit number that has the first and second digit
between 1 and 3

e.g. 11, 12, 13, 21, 22, 23, 31, 32, 33

The first digit is the VERTICAL value
The second digit is the HORIZONTAL value

12 is therefore, the second square on the top row
31 is therefore, the 1st square on the bottom row
etc etc etc
